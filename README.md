## 📖 Overview

AI Lab is an interactive educational platform designed to simplify Machine Learning concepts through visual demonstrations and simulations. It provides intuitive visualizations that help students understand how learning algorithms behave during training and evaluation.

---

## ✨ Features

- Interactive ML visualizations
- Model training simulations
- Underfitting & Overfitting demonstration
- Performance comparison
- Responsive UI
- Modern dashboard

---

## 🛠 Tech Stack

- React
- TypeScript
- Vite
- CSS

---

## 🚀 Installation

```bash
git clone https://github.com/nithinisaimg/ai-lab-master.git
cd AI-Lab
npm install
npm run dev
```
